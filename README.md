# Sarbajit Acharjee portfolio Normal

This is a simple webpage project built using HTML and CSS. It serves as a basic template for anyone looking to create a static webpage.

## Project Structure

- `index.html`: The main HTML file containing the structure and content of the webpage.
- `style.css`: The CSS file for styling the HTML elements.
- `images/`: A directory for storing image assets.

## Getting Started

### Prerequisites

To view and edit this project, you will need:
- A web browser (e.g., Chrome, Firefox, Safari)
